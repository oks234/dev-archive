Gatsby CLI

Install

```
npm install -g gatsby-cli
```

Check

```
gatsby --version
```

Help

```
gatsby --help
```

New

```
gatsby new
```

Run Development Server on local

```
// gatsby-cli가 global 설치된 경우
gatsby develop

// 아닌 경우
npm run develop
```
